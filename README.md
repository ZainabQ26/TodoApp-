### Prerequisites

- Android Studio installed.
- An Android emulator set up in Android Studio, or a physical Android device with developer options enabled.

### Running from Android Studio

1. Open the project in Android Studio.
2. Allow the project to sync with Gradle.
3. Select a device (either an emulator or a connected physical device) from the toolbar.
4. Click the Run 'app' button (the green triangle icon) or press Shift+F10.

### Building from the Command Line

You can also use the included Gradle wrapper to build the app from your terminal.

1. **Build a debug .apk:**

```bash
./gradlew assembleDebug
```

The generated APK can be found in `app/build/outputs/apk/debug/`.

2. **Install the debug app on a connected device:**

```bash
./gradlew installDebug
```
